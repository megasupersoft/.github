## 📌 Pinned Repositories

### [Ghosted](https://github.com/megasupersoft/Ghosted) · TypeScript

> **The dev workspace that never sleeps.**

An agent-native, dark-mode-only development environment with six always-mounted panes, a built-in knowledge graph, and workflow automation — built on Electron, React, and a cold spectral violet accent.

- 🖥️ Monaco editor, real PTY terminal, force-directed knowledge graph
- 🤖 Visual agent workflow canvas powered by `@xyflow/react`
- 📋 Kanban board synced to GitHub Projects v2
- ⚡ Zero-unmount pane model — terminal state and graph positions persist

### [NodeUI](https://github.com/megasupersoft/nodeui) · JavaScript

> **A serverless, node-based visual programming interface for interactive graphs and 3D visualizations.**

A zero-install web application that runs entirely in the browser — no build tools, no dependencies, no server required. Open `index.html` and start building.

- 🔗 Interactive node graph editor with Bezier-curve edges
- 🌐 Full Three.js 3D viewport integration
- 🎞️ Timeline animation system at 24fps
- ☁️ Optional real-time collaboration via Cloudflare Durable Objects

[![License: MIT](https://img.shields.io/badge/license-MIT-white?style=flat-square)](https://github.com/megasupersoft/nodeui/blob/master/LICENSE)
